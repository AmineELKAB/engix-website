Pi5 Web Server for DexiMind

This page provides API endpoints for your Pi5 device that can be accessed directly from the browser.

Server Status: Starting...

Available Endpoints

GET /api/serial - Get Pi5 serial number
GET /api/status - Get device status
GET /api/network - Get network status
GET /api/cpuinfo - Get CPU information

Test Endpoints

Setup Instructions

  1. Save this file on your Pi5 as index.html
  2. Start a simple HTTP server on your Pi5:
# On your Pi5, run:
python3 -m http.server 8080
# or
python -m http.server 8080

Access from Browser

Once running, access these URLs from your computer:

http://YOUR_PI5_IP:8080/api/serial
http://YOUR_PI5_IP:8080/api/status
http://YOUR_PI5_IP:8080/api/network
http://YOUR_PI5_IP:8080/api/cpuinfo
